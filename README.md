This is a Blender plugin and ThreeJs module designed to enhance your 3D design experience. With this tool, you can move objects in Blender, while their position and rotation updates in your ThreeJs scene in real time. This allows you to see how changes will look live. You can have both, Blender and your ThreeJs app, open side by side.

## ðŸš€ Getting Started

### System Requirements

To successfully use threejs-blendertools, ensure you have the following:

- **Blender Version:** Make sure you are using Blender version 2.8 or higher.
- **Node.js and npm:** Install Node.js along with npm for the ThreeJs module.

### Installation Steps

1. **Download Blender Plugin**
   - You can download the Blender plugin from the link below:
     - [Blender Plugin .py](/blender-plugin/threejs-blendertools.py)
   - Place the downloaded `.py` file in your Blender scripts directory.

2. **Install ThreeJs Module**
   - Open your terminal or command prompt.
   - Run the following command to install the module via npm:
     - ```
       npm install threejs-blendertools
       ```

3. **Open Blender**
   - Launch Blender after placing the plugin file.
   - Navigate to the "Edit" menu, then open "Preferences."
   - Find the "Add-ons" section and enable the threejs-blendertools plugin.

4. **Set Up the ThreeJs Scene**
   - Open your ThreeJs application where you want the Blender changes to reflect.
   - Use the features of threejs-blendertools to connect it to Blender.

### How to Use

- Start moving objects in Blender.
- Observe the changes in your ThreeJs scene instantly.
- Adjust settings in the plugin as needed to fit your design process.
